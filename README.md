# Usage

## Package and Install

$ git clone 

$ mvn install

## Maven Dependcy
  ```
<dependency>
  <groupId>org.wltea.analyzer</groupId>
  <artifactId>ik-analyzer</artifactId>
  <version>2012_FF_hf1</version>
</dependency>
```
